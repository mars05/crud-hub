# crud-intellij-plugin

一个用于代码生成的插件，可以根据数据库表结构，帮助您从不同的模板快速生成相关代码。crud-intellij-plugin 插件不仅可以在IDEA中使用，在intellij系列产品中皆可使用。

拥有的功能点:

  • 项目生成: 创建一个新的项目

  • 代码生成: 在已有项目中，根据表结构生成代码

  • 多数据库: 全局维护数据库连接，目前支持MySQL、PostgreSQL、Oracle

  • DDL: 支持从DDL生成代码

  • 模板导入: 按需从模板市场或使用访问令牌导入模板
